What it scans

SQL Injection

Injects test payloads into URL parameters and form inputs to detect backend database vulnerabilities before real attackers do.

XSS (Cross-Site Scripting)

Tests forms and query params with XSS payloads to find reflected/stored script injection vulnerabilities.

SSL / TLS Audit

Checks certificate validity, expiry dates, and weak cipher suites. Flags expired or self-signed certificates immediately.

HTTP Security Headers

Detects missing headers like CSP, HSTS, X-Frame-Options, X-XSS-Protection, and Referrer-Policy.

Open Port Detection

Scans common ports (21, 22, 80, 443, 3306, etc.) to find exposed services that shouldn't be publicly accessible.

WHOIS / DNS Info

Fetches domain age, registrar, DNS records (A, MX, TXT), and flags suspiciously new domains or misconfigs.

Severity rating system

Every vulnerability found is classified and scored. The report groups findings by severity so you know what to fix first.
